# Wrapping utexas-bwi repository with Singularity(https://singularity.lbl.gov/)
## Installation
Singularity requires provides many installation options.(https://singularity.lbl.gov/install-linux)  
I recommand to use below command to install Singularity in Ubuntu.
* Note that you SHOULD use Ubuntu 18.04 or above. Below installation code does not work with Ubuntu 16.04.
$ sudo apt-get update
$ sudo apt-get install -y singularity-container

## Make Image for the singularity
Singularity requires SUDO previlage for image generation process. Please follow instructions with a machine you have a sudo previlage.


### Develop
Installation process should run by itself. If any of process requires user interation, singularity will pause building process and spit 255 error.
